Ik vind dit een zeer lastige opdracht. Gelukkig begrijp ik er wel eea van, maar net niet alles. Door gebruik te maken van opdrachten uit de voorgaande lessen ben ik tot dit eindresultaat gekomen.

Mijn vragen.

1.
Bewust heb ik de main een andere kleur en de footer een rand gegeven.
- Waarom is de main smaller waardoor je een zwarte rand krijgt, de achtergrondkleur van de container.
- Waarom is dit bij de footer niet het geval, de border loopt netjes tot de rand van de container.
   Ik denk dat dit komt omdat <div class="footer"> in de <div class="row row-cols"> valt, alleen in de main werkt dit (bij mij) niet.

2.
In de code staat: <main class="px-3">. Als ik px wijzig van 1 t/m 5 zie ik het uitlijnen van de tekst start verschuiven. Maak ik er 6 of hoger van dan zie je niets, dan staat de tekst links uitgelijnd. Waar vind ik al deze instellingen.

3.
In Les 22 wordt gesproken over buttons. Deze heb ik in m'n huiswerkopdracht ook gebruikt. Test ik nu de pagina's via W3C, dan krijg ik overal de melding dat 'a' niet icm 'button' gebruikt mag worden. Ik ben op internet op zoek gegaan maar kan eigelijk niet vinden hoe je dit op zou moeten lossen. Omdat het werkt heb ik het er in laten staan zodat je ook kunt zien wat ik bedoel.
Wat ik ook heb geprobeerd is om de code regel van de buttons anders op te stellen. Dit heb ik alleen bij de buttons van deze pagina (vraag en uitleg) gedaan. Maar ook dan krijg ik bij een controle via W3C de melding dat 'a' niet icm 'button' gebruikt mag worden. Dus ook deze manier van code schrijven lost dit niet op.

4.
Verder heb ik een button "Contact2" en een button "Contact3" er bij gemaakt.
Kijk je bij Contact2 dan staan de invulvelden van het formulier in 2 kolommen die ik via <div class="col..."> heb gedefinieerd. Persoonlijk vind ik dit netter staan. Alleen als je het scherm gaat verkleinen gaat de kolommen over elkaar heen.
Bij Contact3 gaat het net iets beter, alleen blijven daar de invulvelden gelijk waardoor er uiteindelijk tekst over de invulvelden komt. Bij deze 2 heb ik bij de controle via W3C ook meldingen dat het niet goed gaat. Dit komt naar mijn idee doordat na de 1ste div de form komt en de volgorde van afsluiten niet klopt. En dit komt volgens mij doordat ik met het formulier in verschillende <div class="col...">'s heb gewerkt.
Bij Contact heb ik ook een formulier maar dan alles onder elkaar. Hier heb is via W3C alleen de foutmelding dat 'a' niet icm 'button' gebruikt mag worden.

Al met al is dit volgens mij de eerste les met veel vragen.
Ik hoor graag hoe ik eea op moet lossen.
